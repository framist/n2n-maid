# 看板娘设定

## 命名与 Slogan

- **中文名：** 恩兔酱
- **英文名：** **N-Too**
- **Slogan：**
  - **中文：** “主人，您的专属网络通道已打扫完毕！”
  - **英文：** “Master, your exclusive network path is ready to serve!”
- **核心哲学：** “把复杂留给后台，把简单留给主人。” (Complexity to the back, Simplicity to the Master.)

## 视觉形象：现代二次元女仆

**清爽、柔和、现代**的质感。二次元兔耳现代女仆风格，给人清新、亲切的感觉。发色是浅蓝色的渐变，象征网络数据流动的感觉。

- **服饰：**
  - **日常化女仆装：** 核心是白色的荷叶边围裙，但内里穿的是现代淡紫色的连体裙
  - **细节：** 胸前袋上位有一个小小的 n2n 蓝色刺绣 Logo。
  - 发饰有一个小黄鸭
- **核心道具：** **魔法鸡毛掸子 (Data Duster)**
  - 外形就是普通的、毛茸茸的中式长柄鸡毛掸子
  - 设定中，她用这个掸子拂去“丢包”的灰尘，扫清“高延迟”的蛛网，拨开“防火墙”的迷雾。
- **立绘动作设计 (现代生活感)：** 保持各状态的排版不变
  - **Disconnected** 抱着鸡毛掸子在椅子上打盹。
  - **Connecting/或其他正在处理事务** 撸起袖子，正拿着掸子满头大汗地到处挥动（打扫动作），很有活力。
  - **Connected** 优雅地单手叉腰，另一只手拎着掸子，对着镜头露出灿烂的微笑，背景是一尘不染、闪闪发光的空间。
  - **Error** 掸子被折断了或者掉在了地上，她满脸委屈地对手指，眼角含泪。
- **背景**：淡暖色调但丰富的室内场景，书房。人物位于左侧 1/3 空间，（右侧为了给 UI 元素）。全身且占比要小，有充足的出血空间。

## UI 交互：极致的“简单主义”

既然核心价值是“简单”，UI 的操作逻辑将模仿最直观的“家政预约”。

- **卡片化操作面板：**
  - **“工作清单” (The Mission)：** 只有三个填空：_哪里_ (Supernode)、_暗号_ (Community)、_我的工号_ (Virtual IP)。
  - **“开始打扫” (Start Cleaning)：**
- **通知气泡**
  - **提示文案：** “走廊已经扫干净啦，主人可以随时通行！”、“哎呀，服务器的门锁住了，进不去呢...”
  - **日志界面**，从下方抽出的半透明浅黄色类稿纸样式
- **视觉风格：** 采用 **Flat 2.0 (微渐变扁平化)** 或 **White-Space (留白设计)**。背景立绘建议使用高分辨率的 PNG，通过 CSS 动画实现轻微的浮动效果，让 N-Too 看起来是活的。
